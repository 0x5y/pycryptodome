__init__(self, asn1Id=None, payload='', implicit=None, constructed=False, explicit=None)
(Constructor)

 
Initialize the DER object according to a specific ASN.1 type.
Parameters:
  • asn1Id (integer) - The universal DER tag number for this object (e.g. 0x10 for a SEQUENCE). If None, the tag is not known yet.
  • payload (byte string) - The initial payload of the object (that it, the content octets). If not specified, the payload is empty.
  • implicit (integer) - The IMPLICIT tag number to use for the encoded object. It overrides the universal tag asn1Id.
  • constructed (bool) - True when the ASN.1 type is constructed. False when it is primitive.
  • explicit (integer) - The EXPLICIT tag number to use for the encoded object.
